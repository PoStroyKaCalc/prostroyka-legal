Политика конфиденциальности

для мобильного приложения «ПоСтройКа»

1. Кто мы

Разработчик приложения: ООО «Проф-Решения».

Контакты поддержки: postroykacalc@yandex.com.

2. Какие данные обрабатываются

Приложение «ПоСтройКа» рассчитано на автономную работу: расчёты и введённые пользователем значения обрабатываются на устройстве.

В зависимости от используемых функций, приложение может обрабатывать следующие данные:

  • Геолокация — используется для формирования поисковых запросов материалов и предложений в разделе «В магазин» (поиск рядом с пользователем). Геолокация запрашивается только при необходимости и в объёме, предусмотренном разрешениями устройства.
  • Данные, введённые пользователем (параметры расчёта, реквизиты/телефон для документов, заметки) — используются для формирования результатов и документов. По умолчанию хранятся локально на устройстве пользователя.
  • Технические данные устройства (версия ОС, модель устройства) — могут использоваться для обеспечения совместимости и устранения ошибок, если такие средства будут включены в приложении.

3. Чего мы не делаем

  • Мы не продаём ваши персональные данные третьим лицам.
  • Мы не собираем “лишние” данные, не необходимые для работы приложения.

4. Цели обработки

  • Работа калькуляторов и формирование результатов расчёта.
  • Формирование документов (PDF) по запросу пользователя.
  • Поиск материалов в разделе «В магазин» с учётом местоположения пользователя (если предоставлено разрешение).
  • Поддержка пользователей и ответы на обращения.

5. Где хранятся данные

Расчёты, сохранения и введённые значения по умолчанию хранятся локально на устройстве пользователя. Отправка данных во внешние сервисы возможна только в рамках функций, инициированных пользователем (например, «Поделиться» документом через выбранное приложение).

6. Разрешения

Приложение может запрашивать разрешения, необходимые для конкретных функций (например, доступ к геолокации для «В магазин»). Вы можете отозвать разрешение в настройках устройства; часть функций может работать ограниченно.

7. Передача третьим лицам

Мы можем передавать данные третьим лицам только в случаях, необходимых для выполнения функций приложения (например, открытие внешнего браузера/сервиса для поиска материалов) и/или предусмотренных законодательством.

8. Безопасность

Мы применяем разумные меры защиты данных. Однако абсолютная безопасность в интернете не может быть гарантирована.

9. Ваши права

Вы можете:

  • управлять разрешениями приложения в настройках устройства,
  • удалить данные приложения, удалив приложение или очистив его данные в настройках ОС,
  • обратиться в поддержку по вопросам данных: postroykacalc@yandex.com.

10. Изменения политики

Мы можем обновлять эту Политику. Актуальная версия всегда доступна по этой ссылке.
